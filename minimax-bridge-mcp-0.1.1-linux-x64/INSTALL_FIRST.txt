  一键安装指南
═══════════════════════════════════════════════════════════════

前提条件：
  - 已安装 Node.js 20 或更高版本 (https://nodejs.org/)

Windows 用户：
  双击运行 install.bat，然后按提示输入 API Key 即可。

macOS / Linux 用户：
  打开终端，执行以下命令：
    chmod +x install.sh
    ./install.sh
  然后按提示输入 API Key 即可。

安装完成后：
  重启 OpenCode，即可使用 MiniMax 的各种 AI 工具。

这是一个 stdio MCP 服务器，OpenCode 会自动启动和管理它。

详细文档请查看 README.md
